CSS
i. Internal CSS
- it is written inside the <head> </head> tag of the html page
example:
<head>
<style>
selector{
properties: value;
}
</style>
</head>

ii. external CSS
- it is written in the external CSS page and the CSS page has a file extraction dot CSS (.CSS)
- to link external CSS with the html page do the following 
syntax:
<head>
<link rel="stylesheet" href="filename.css">
</head>

iii. inline CSS:
-it is written in the tag as the attribute
<p style="property1: value1, property2:value2;">......</p>
